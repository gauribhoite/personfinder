delete module¶

class delete.Handler(request, response, env)¶

Bases: utils.BaseHandler

Handles a user request to delete a person record.

get()¶

Prompts the user with a Turing test before carrying out deletion.

post()¶

If the user passed the Turing test, delete the record.

delete.delete_person(handler, person, send_notices=True)¶

Delete a person record and associated data. If it’s an original record, deletion can be undone within EXPIRED_TTL_DAYS days.

delete.get_restore_url(handler, person, ttl=259200)¶

Returns a URL to be used for restoring a deleted person record. The default TTL for a restoration URL is 3 days.

delete.get_tag_params(handler, person)¶

Return HTML tag parameters used in delete.html.

delete.send_delete_notice(handler, person)¶

Notify concerned folks about the potential deletion.

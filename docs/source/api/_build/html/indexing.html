indexing module¶

Support for approximate string prefix queries.

A hit is defined when the words entered in the query are all prefixes of one of the words in the given and family names on the record. For example, a record with the fields:

given_name: ABC 123 family_name: DEF 456
will be retrieved by:
“ABC 456” “45 ED” “123 ABC” “ABC 123 DEF”
but will not be retrieved by:
“ABC 1234” “ABC 123 DEF 456 789”

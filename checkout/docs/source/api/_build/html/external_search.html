external_search module¶

external_search.fetch_with_load_balancing(urls, fetch_timeout=1.0, total_timeout=5.0)[source]¶

Attempt to fetch a content from one or more urls.

Args:
urls: A list of urls from which content may be fetched. This may be
iterated over several times if needed.
fetch_timeout: The time in seconds to allow for one request before
retrying.
total_timeout: The total time in seconds to allow for all requests
before giving up.
Returns:
A urlfetch.Response object, or None if the timeout has been exceeded.

external_search.search(repo, query_obj, max_results, backends)[source]¶

Search persons using external search backends.

Args:
repo: PF’s repository for which the query was sent. query_obj: TextQuery instance representing the input query. max_results: Maximum number of entries to return. backends: List of backend IPs or hostnames to access.
Returns:
List of Persons that are returned from an external search backend (may be []), or None if backends return bad responses.
